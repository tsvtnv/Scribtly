## ✨ What is Scribtly?

Scribtly is an AI-powered script generator built for content creators — YouTubers, TikTokers, Instagram Reels creators, and short-form video producers. Generate platform-specific scripts with the right hooks, tone, and pacing in seconds.

## 🔥 Features

- **🎬 Script Generator** — Pick platform, hook style, tone, length — AI does the rest
- **📚 Script Library** — Save, search, and organise all your generated scripts
- **🎯 Platform Optimised** — Tailored for YouTube, TikTok, Reels, and Shorts
- **🤖 Multi-Model AI** — Powered by OpenAI GPT and Anthropic Claude

## 🛠 Tech Stack

| Layer | Technology |
|---|---|
| Framework | Next.js (App Router) |
| Language | TypeScript |
| Styling | Tailwind CSS |
| Database | Prisma ORM |
| AI | OpenAI · Anthropic Claude |
| Deploy | Vercel · Nixpacks |

## 🚀 Getting Started

```bash
git clone https://github.com/tsvtnv/Scribtly.git
cd Scribtly
npm install
cp .env.example .env.local
npx prisma generate
npm run dev
```
